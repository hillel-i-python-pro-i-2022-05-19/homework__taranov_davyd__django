# Project on a Django 

___

## Docker

### Run

To run the app in docker use:

```shell
make d-run
```

#### After, go to [http://localhost:8000](http://localhost:8000)

### Stop

To stop app in docker use:

```shell
make d-stop
```

### Clean

To clean up an app in docker use:

```shell
make d-purge
```
